### 🛠️ Personal Projects

#### 🚛 [TruckLogs](https://www.trucklogs.tech/) — Backhaul Logistics Platform
A digital platform that connects truckers with available cargo space on return trips to companies needing freight, reducing logistics costs by up to 20%.

**Key Features:**
- QR-based load verification and real-time geofencing tracking
- Digital proof of delivery with electronic signatures and ID verification
- Built-in quote & negotiation system between truckers and companies
- Automatic payment settlement upon delivery confirmation
- Automated document verification (VIG, insurance, driving licenses)

**Stack:** Next.js · TypeScript · PostgreSQL · AWS

#### ⛓️ [TokenForge](https://alejandro04.github.io/TokenForge/) — Open Source Blockchain Asset Tokenizer
An open-source Python CLI tool to deploy ERC-20 and ERC-721 tokens on any EVM blockchain in under 60 seconds — no dashboards, no fees, no vendor lock-in.

**Key Features:**
- Interactive 5-question wizard that deploys token contracts directly from the terminal
- Multi-chain support: Ethereum, Polygon, Arbitrum, Base + custom RPC
- Supports ERC-20 (fungible) and ERC-721 (NFT) standards, with ERC-3643 and ERC-1155 on the roadmap
- Auditable Solidity contracts based on OpenZeppelin patterns, verifiable on Etherscan
- MIT licensed — completely free, just pay blockchain gas

**Stack:** Python · Solidity · EVM chains

🔗 [GitHub Repository](https://github.com/Alejandro04/TokenForge)
